How To Play Killer Sudoku:

For rules: https://en.wikipedia.org/wiki/Killer_sudoku

How to play here:

  • if you want to insert a number in any cell;click that cell(not near edges). blinking cursor will apear on that cell. insert the number. same way fill the board.
  • if you want to change any cell's value;click on that cell(not near edges). blinking cursor will apear on that cell. insert the number and value of that cell will change.
  • whenever you want to verify if answer is true or not; press "Click Here to verify".ONLY IF EACH CELL IS CORRECT;prompt message will come saying "Congratulations ! correct answer" otherwise it will show "you lose,wrong answer".
  • clicking okay on prompt dialogue box;you will return to webpage with same values in the cell those were before clicking verify button
  • to reload puzzle press "Reload"

Note: user may insert character those are not between 1-9;even though you can see inserted character on that cell;while verifying as EACH CELL will not have CORRECT number;it will show "you lose,wrong answer".
